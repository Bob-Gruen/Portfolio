Agent Learns Using Gentic Algorithm

This agent learns to play using a genetic algorithm. 50 cars are initially spawned (using random weights in their brains). After a few moments, save the best car (click Save button). Now restart the simulation (clicking the link in the left nav), and 50 of the new cars will spawn using the saved best car. Repeat the cycle and the performance improves over time.

Key skills used in this project include: JavaScript, Genetic Algorithms, CSS, HTML
